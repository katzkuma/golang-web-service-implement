# Golang-web-service-implement
### Environment-Info
Language: Golang
Web-framework: Gin
Database: MySQL＠8.0.21

### Steps of building environment:
1. brew install go
2. go get -u github.com/gin-gonic/gin
3. go get "github.com/go-sql-driver/mysql"
